Workflow erstellen

Rezept erstellen

wineq_rec <- recipe(quality ~ alcohol, volatile.acidity,
                        data = wineq_train) %>%
  step_normalize(all_predictors())

Modell definieren

Ich erstelle ein kNN-Modell zur Regression und tune im spteren Verlauf k

knn_model <-
  nearest_neighbor(
    mode = "regression",
    neighbors = tune()
  )

Workflow definieren

wineq_wflow <- 
  workflow() %>% 
  add_model(knn_model) %>% 
  add_recipe(wineq_rec)

Tuning

Erstellen der Modelle

Ich erstelle ein tibble fr die k werte

knn_grid <-  tibble(neighbors = 1:10)

Jetzt berechne ich die Modelle

knn_results <- tune_grid(
  wineq_wflow,
  resamples = vfold_cv(wineq_train, strata = "quality"),
  grid = knn_grid)

Abschlieende Gedanken

Abschlieend lsst sich sagen, dass das man bestimmt ein besseres Regressionsmodell auf die Beine stellen knnte, aber sicherlich auch ein schlechteres. Ein wichtiger Gedanke hierbei ist die Frage, ob ein Regressionsmodell berhaupt am besten geeignet ist fr diesen Datensatz. Ich kann mir vorstellen, dass ein Klassifikationsmodell genauso gut mglich ist, wenn nicht sogar besser.
